🌐 LLM Assistant 多语言测试页面

🎉 新功能：支持任何语言

现在复制任何语言的内容都会弹出功能区，智能推荐最合适的功能！

测试说明：
  1. 确保已安装并启用插件
  2. 点击下方文本框复制内容
  3. 观察功能区是否出现，以及推荐功能是否合理
  4. 尝试不同语言的内容
  5. 注意语法检查功能仅对英文内容启用
